# 🪔 Diwali Sales Analysis | Exploratory Data Analysis (EDA)

## 📌 Project Overview

This project analyzes a Diwali Sales dataset to identify key customer segments, revenue-driving states, and high-performing product categories using Python-based data analysis and visualization techniques.

---

## 📊 Dataset Summary

- Total Records: 11,251
- Columns (after cleaning): 13
- Data Type: Structured CSV dataset
- Key Features: Gender, Age Group, Marital Status, State, Occupation, Product Category, Orders, Amount

---

## 🛠 Tools & Technologies

- Python
- Pandas (Data Cleaning & Aggregation)
- NumPy
- Matplotlib & Seaborn (Visualization)
- Jupyter Notebook

---

## 🧹 Data Preprocessing

- Removed irrelevant columns
- Handled missing values (12 rows removed)
- Converted data types for accurate numerical analysis
- Performed grouping, sorting, and aggregation using `groupby()`

---

## 📈 Key Analytical Findings

### 👩 Gender Insights
- Female customers contributed significantly higher total sales than males.

### 👥 Age Group Insights
- Customers aged **26–35 years** showed the highest purchasing activity and revenue contribution.

### 🌍 Top Revenue States
- Uttar Pradesh
- Maharashtra
- Karnataka

### 💍 Marital Status
- Married customers, especially women, generated higher sales.

### 💼 Occupation
- IT, Healthcare, and Aviation professionals showed stronger purchasing power.

### 🛍 Product Categories
- Food
- Clothing & Apparel
- Electronics & Gadgets  
were the top-performing categories during the festive season.

---

## 🎯 Business Insight

The most profitable customer segment is:

**Married women aged 26–35 years from major revenue-generating states working in professional sectors.**

This insight can help businesses optimize:
- Targeted marketing campaigns
- Festive offers
- Inventory planning
- Regional promotions
